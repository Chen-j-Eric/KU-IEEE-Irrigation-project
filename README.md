# KU-IEEE Irrigation Project

This repository contains the source code and documentation for the KU IEEE Irrigation Project. **Arduino IDE v2.3.6** with the **Espressif ESP32 plugin v3.3.2**.

This project was selected as **1 of 5 projects** from a pool of **25 proposals**, fundded by **KU IEEE**.

**Special thanks to:**  
**Noah Hickman** and **Joe Lieber** for being apart of the team!

## Parts List
- Freenove ESP32 Board  
- 12" Plant Container with Drainage Hole  
- 4× DC Motors  
- 5V 4-Channel Relay Module  
- Gravity IP65 Capacitive Soil Moisture Sensor  
- 5-Herb Seed Variety Kit
- Water tank level sensor (to prevent motor damage when the tank is empty)
- Final water tank size 6x6x6
