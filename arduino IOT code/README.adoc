= Project: IEEE-irrigation-project

== Step 1: Installation
1. Make a account for: https://cloud.arduino.cc/
2. Connect your ESP 32: https://docs.arduino.cc/arduino-cloud/guides/esp32/
3. Make your variables: https://docs.arduino.cc/arduino-cloud/cloud-interface/variables/
3. Set up your Dashboard: https://docs.arduino.cc/arduino-cloud/cloud-interface/dashboard-widgets/

Note: 
If you are having issues, it is likely that your dashboard variables do not mach the ones in the code.
Replace it with the ones you have or change it to what the code is written to do.

// Name of the buttons: (They are cap senstiive)
void onWaterOn1Change();
void onWaterOn2Change();

// Name of the variables: (They are cap senstiive) 
int soil_Dryness_1;
int soil_Dryness_2;
bool float_Sensor;
bool water_On_1;
bool water_On_2;



== Step 2: Assemble the circuit

Picture of the circuit are on the Github, follow the define pins marked on the ESP32.

== Step 3: Load the code

Copy and paste the ESP32 code into your IDE.
